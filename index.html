How to Use the VLOOKUP() Function in Excel

Short Tutorial by Sydney Doyle



What is VLOOKUP?


The VLOOKUP, or vertical lookup, allows you to search for unknown information using known information. For example, you might use VLOOKUP to find a client’s subscription plan based on their company name.

VLOOKUP is especially useful when your spreadsheet contains many rows and columns that make it difficult to search for a specific value manually. You will use this function often if you need to make repetitive inqueries within the same spreadsheet.

Syntax: =VLOOKUP(lookup_value,table_array,col_index_num,[range_lookup]

lookupvalue: The value you want to look up
table_array: The search range beginning with the column where the initial value is located
col_index_num: The column number where the return value is located
range_lookup: 0 or FALSE for an exact match, 1 or TRUE for an approximate match

Lesson


  • This is a guide to Microsoft Excel’s VLOOKUP() function, which lets you search for unknown information using known information.

  • Let’s use this example worksheet to find out what company WBR pays per station. For the purposes of this example, this dataset is quite small but imagine a dataset with many columns and rows that would be too difficult to search by eyesight.

  • First we want to determine the value to look up. So, we would type in the name “WBR” in quotations OR type the value into a cell and select it.

  • Either works!

  • Next, we need to enter the table array or the search range.

  • Imagine the VLOOKUP() function’s movement in the shape of an L.

  • First, it’s going to search down the column to find the initial value you entered. Then, it will search across the row to find the return value.

  • Therefore, our search range should span the company data to the end of the data set. Your table array should always begin with the column where the initial value is located.

  • Then, we must enter the column number where our return value is located. Since we want to find out the price per station, our column number is 4.

  • Finally, we need to decide whether we’re looking for an exact or approximate match. If we need an exact match, we would enter the number 0 or FALSE. If we need an approximate match, we would enter the number 1 or TRUE.

  • Since we know the exact company name that we’re looking up, we can search for an exact match.

  • Now we can see that WBR pays $2,400 per station.

    Answer: Professional
    Formula: =VLOOKUP("Caza Thocolate",A2:E11,2,0)
