How to Use the VLOOKUP() Function in Excel

Short Tutorial by Sydney Doyle



What is VLOOKUP?


The VLOOKUP(), or vertical lookup, is a search function in Excel that allows you to find things in a table or a range by row. For example, you might use VLOOKUP to find a clientâ€™s subscription plan based on their company name.

VLOOKUP is especially useful when your spreadsheet contains many rows and columns that make it difficult to search for a specific value by eyesight alone.

Syntax: =VLOOKUP(lookup_value,table_array,col_index_num,[range_lookup]

lookupvalue: The value you want to look up
table_array: The search range beginning with the column where the initial value is located
col_index_num: The column number where the return value is located
range_lookup: 0 or FALSE for an exact match, 1 or TRUE for an approximate match
